## 快速导航

### 什么是大模型？

大模型（LLM）是一种基于深度学习的 AI 模型，能够理解和生成人类语言。

```python
response = openai.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "你好"}]
)
```

### 什么是 AI Agent？

AI Agent 是能够**自主感知、决策、执行**的智能系统，不只是回答问题，还能使用工具、规划任务。

```
普通 AI：用户提问 → AI 回答
AI Agent：用户目标 → 规划 → 使用工具 → 执行 → 完成目标
```

### Agent 核心组成

```
┌─────────────────┐
│   大模型(LLM)   │  ← 大脑：理解、推理、决策
├─────────────────┤
│     规划器      │  ← 计划：分解任务、制定步骤
├─────────────────┤
│    记忆系统     │  ← 记忆：存储历史、上下文
├─────────────────┤
│     工具集      │  ← 能力：搜索、计算、API调用
└─────────────────┘
```

## 核心知识点速查

### Prompt 技巧

| 技巧 | 说明 |
|------|------|
| 角色设定 | 给 AI 一个明确的身份 |
| Few-shot | 提供示例帮助理解 |
| Chain of Thought | 让 AI 一步步思考 |
| 结构化输出 | 指定输出格式 |

### 工具调用流程

```
用户请求 → AI 判断需要工具 → 选择工具生成参数 → 执行工具 → 返回结果 → AI 整合回答
```

### 记忆类型

| 类型 | 用途 | 实现 |
|------|------|------|
| 短期记忆 | 当前对话 | 消息列表 |
| 长期记忆 | 持久知识 | 数据库存储 |
| 向量记忆 | 语义搜索 | 向量数据库 |

### 主流框架

| 框架 | 特点 |
|------|------|
| LangChain | 生态丰富，通用性强 |
