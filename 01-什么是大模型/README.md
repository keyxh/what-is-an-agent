# 什么是大模型 (LLM)

## 简单定义

大模型（Large Language Model，简称 LLM）是一种基于深度学习的人工智能模型，通过海量文本数据训练，能够理解、生成和处理人类语言。

## 核心概念

### 1. 什么是"大"？

- **参数量大**：现代大模型通常有数十亿到数千亿个参数
  - GPT-3: 1750 亿参数
  - GPT-4: 参数量未公开，估计更大
  - Claude: 参数量未公开
  - Llama 2: 70 亿到 700 亿参数

- **训练数据量大**：使用互联网上的海量文本进行训练
  - 书籍、文章、网页、代码等

### 2. 工作原理

```
输入文本 → 模型处理 → 预测下一个词 → 输出文本
```

大模型本质上是"预测下一个词"的机器，通过不断预测来生成连贯的文本。

### 3. 核心能力

| 能力 | 说明 | 示例 |
|------|------|------|
| 文本生成 | 生成连贯的文章、故事、代码 | 写一篇文章、生成代码 |
| 文本理解 | 理解文本含义和上下文 | 阅读理解、摘要 |
| 问答对话 | 回答问题、进行对话 | 聊天机器人 |
| 推理能力 | 进行逻辑推理 | 数学问题、逻辑题 |
| 多语言 | 支持多种语言 | 翻译、多语言对话 |

### 4. 什么是多模态？

传统大模型只处理文本，而**多模态大模型**可以处理多种类型的信息：

```
┌─────────────────────────────────────────────────────────────┐
│                     多模态大模型                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   单模态（仅文本）          多模态（文本 + 更多）             │
│   ┌─────────────┐          ┌─────────────────────┐         │
│   │   文本输入   │          │  文本 │ 图像 │ 音频 │         │
│   │      ↓      │          │   ↓     ↓     ↓    │         │
│   │   文本输出   │          │    统一模型处理     │         │
│   └─────────────┘          │         ↓          │         │
│                            │  文本/图像/操作输出  │         │
│                            └─────────────────────┘         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 模态类型

| 模态 | 说明 | 应用场景 |
|------|------|----------|
| 文本 | 文字输入输出 | 对话、写作、翻译 |
| 图像 | 图片理解与生成 | 图像描述、OCR、视觉问答 |
| 音频 | 语音识别与合成 | 语音助手、转录 |
| 视频 | 视频理解 | 视频摘要、内容分析 |

#### 主流多模态模型

| 模型 | 支持模态 | 特点 |
|------|----------|------|
| GPT-4o | 文本、图像、音频 | 最强多模态能力 |
| Gemini | 文本、图像、视频、音频 | 原生多模态设计 |
| Claude 3 | 文本、图像 | 图像理解优秀 |
| Qwen-VL | 文本、图像 | 中文友好，开源 |
| LLaVA | 文本、图像 | 开源，可本地部署 |

#### 多模态 vs 单模态

```
单模态场景：
用户："什么是猫？"
AI：用文字描述猫的特征

多模态场景：
用户："这张图片里有什么？" + [猫的图片]
AI：识别图片内容，描述"这是一只橘色的猫，正在阳光下睡觉"
```

## 大模型 vs 传统程序

```
传统程序：
输入 → 固定规则 → 输出
（确定性，可预测）

大模型：
输入 → 神经网络 → 输出
（概率性，创造性）
```

## 如何使用大模型

### 1. API 调用

```python
import openai

response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[
        {"role": "user", "content": "你好"}
    ]
)
print(response.choices[0].message.content)
```

### 2. 网页界面

- ChatGPT: https://chat.openai.com
- Claude: https://claude.ai
- 文心一言: https://yiyan.baidu.com

### 3. 本地部署

使用开源模型如 Llama、ChatGLM 等可以在本地部署运行。

## 大模型的局限性

1. **幻觉问题**：可能生成不真实的信息
2. **知识截止**：训练数据有时间限制
3. **数学能力**：复杂计算可能出错
4. **上下文长度**：输入长度有限制
5. **成本**：使用 API 可能产生费用
